# 📊 Monthly Wafer Capacity Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** that analyzes the **Monthly Wafer Capacity** of major semiconductor manufacturers. The dashboard enables users to explore production capacity trends across companies, fabrication types, and years, providing valuable insights into the semiconductor industry's manufacturing landscape.

---

## 🎯 Objectives

- Analyze monthly wafer production capacity by company.
- Compare production capacities across leading semiconductor manufacturers.
- Explore wafer capacity trends over time.
- Analyze capacity distribution by fabrication (Fab) type.
- Build an interactive dashboard for business insights.

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**
- **Interactive Visualizations**

---

## 📈 Dashboard Features

### 1. Company Filter
- Interactive slicer to filter the report by semiconductor company.

### 2. Company-wise Monthly Wafer Capacity
- Horizontal bar chart showing wafer capacity comparison across companies.

### 3. Fab Type Distribution
- Donut chart displaying wafer capacity distribution by fabrication type:
  - Logic Mature
  - Logic Leading
  - Memory DRAM
  - Memory NAND
  - Memory HBM Packaging
  - Logic Mature 300mm

### 4. Year-wise Capacity Trend
- Area chart illustrating the growth of monthly wafer capacity over the years.

### 5. Interactive Filters
- Company Name
- Fab Type
- Year

All visuals are interconnected, allowing dynamic filtering and exploration.

---

## 📊 Key Insights

- **TSMC** has the highest monthly wafer production capacity among the selected companies.
- Memory manufacturers such as **Samsung Memory** and **SK Hynix** contribute significantly to global wafer production.
- Logic mature fabrication accounts for the largest share of total wafer capacity.
- Wafer production capacity has shown a steady upward trend over the years.
